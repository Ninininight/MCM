| **语法**                            | **说明**                                                                                                    | **输入参数**                                                     |
| --------------------------------- | --------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------ |
| `G = graph`                       | 创建一个**空的**无向图对象。                                                                                          | 无                                                            |
| `G = graph(A)`                    | 使用**邻接矩阵** `A` 创建赋权无向图。对于无向图，`A` 必须是对称矩阵（或使用 `'upper'` / `'lower'` 选项）。                                   | `A`: 邻接矩阵                                                    |
| `G = graph(A, nodes)`             | 使用**邻接矩阵** `A` 和**节点名称** `nodes`（通常是字符串数组或元胞数组）创建赋权无向图。                                                   | `A`: 邻接矩阵, `nodes`: 节点名称                                     |
| `G = graph(s, t)`                 | 使用**顶点对组** `s` (起点) 和 `t` (终点) 创建无向图。`s` 和 `t` 是等长的向量，对应每条边。边的权重默认为 1。                                    | `s`, `t`: 起点和终点向量                                            |
| `G = graph(s, t, weights)`        | 使用**顶点对组** `s, t` 和**权重向量** `weights` 创建赋权无向图。                                                            | `s`, `t`: 起点和终点向量, `weights`: 权重向量                           |
| `G = graph(s, t, weights, nodes)` | 使用**顶点对组** `s, t`、**权重向量** `weights` 和**字符串元素数组** `nodes` 来指定**顶点名称**。                                    | `s`, `t`: 起点和终点向量, `weights`: 权重向量, `nodes`: 顶点名称            |
| `G = graph(s, t, weights, num)`   | 使用**顶点对组** `s, t`、**权重向量** `weights` 和**数值** `num` 来指定图中的**顶点数**。如果 `s` 和 `t` 中最大的索引小于 `num`，则会创建额外的孤立顶点。 | `s`, `t`: 起点和终点向量, `weights`: 权重向量, `num`: 顶点总数              |
| `G = graph(A, [nodes], type)`     | 仅使用邻接矩阵 `A` 的**上三角形**或**下三角形**构造赋权图。`type` 可以是 `'upper'` 或 `'lower'`。这在处理冗余的对称邻接矩阵时很有用。                   | `A`: 邻接矩阵, `nodes`: (可选) 节点名称, `type`: `'upper'` 或 `'lower'` |



|**参数**|**作用**|**示例**|
|---|---|---|
|`'EdgeLabel'`|在边旁边显示标签（如权重）。|`plot(G, 'EdgeLabel', G.Edges.Weight)`|
|`'NodeLabel'`|在节点旁边显示标签（如节点名称或索引）。|`plot(G, 'NodeLabel', {'A', 'B', 'C', 'D'})`|
|`'Layout'`|指定绘图布局算法。常见有 `'auto'`, `'force'`, `'circle'`, `'subspace'` 等。|`plot(G, 'Layout', 'force')`|
|`'NodeColor'`|指定节点的颜色。|`plot(G, 'NodeColor', 'r')`|
|`'Marker'`|指定节点的形状。|`plot(G, 'Marker', 's')`|
|`'XData'`, `'YData'`|**自定义**节点的 $x$ 和 $y$ 坐标。|`plot(G, 'XData', [1 1 2 2], 'YData', [1 2 1 2])`|



```matlab
%% 图论基础操作：创建、绘制有向图及导出矩阵

% 1. 环境初始化
clc;        % 清空命令行窗口
clear;      % 清除工作区所有变量
close all;  % 关闭所有图形窗口

%% 2. 数据定义 (边列表)
% 定义有向边的列表 E = [起点, 终点]
E = [1, 2;
     1, 3;
     2, 3;
     3, 2; % 注意：此边方向与 2->3 相反
     3, 5;
     4, 2;
     4, 6;
     5, 2;
     5, 4;
     6, 5];

% 分离起点 (Source) 和终点 (Target) 向量
s = E(:, 1);
t = E(:, 2);

% 动态生成节点名称 (v1, v2, ..., v6)
% cellstr: 转换为元胞数组
% strcat('v', ...): 拼接字符串 'v' 和节点索引
% int2str([1:6]'): 1到6的数字转换为列字符串
nodes = cellstr(strcat('v', int2str([1:6]')));

%% 3. 创建有向图对象
% G = digraph(起点向量, 终点向量, 权重向量[], 节点名称)
% [] 表示创建未赋权图
G = digraph(s, t, [], nodes);

disp('--- 有向图对象 G 信息 ---');
disp(G);
disp('--------------------------');

%% 4. 绘制有向图
figure; % 创建新的绘图窗口
plot(G, ...
    'LineWidth', 1.5, ...       % 设置边的宽度
    'Layout', 'circle', ...     % 使用圆形布局 (circular layout)
    'NodeFontSize', 15);        % 设置节点标签的字体大小

title('有向图 (Digraph) 可视化 - 圆形布局');

%% 5. 导出图的矩阵表示
% W1 和 W2 默认为 MATLAB 的稀疏矩阵格式，以节省内存

% 导出邻接矩阵 (Adjacency Matrix) 的稀疏矩阵
% W1(i, j) = 1 表示存在从节点 i 到节点 j 的边
W1 = adjacency(G);
disp('邻接矩阵 (W1) 的稀疏表示:');
disp(full(W1)); % 使用 full() 查看完整的稠密矩阵

% 导出关联矩阵 (Incidence Matrix) 的稀疏矩阵
% W2(i, k) = -1 (边 k 从节点 i 发出); W2(j, k) = 1 (边 k 进入节点 j)
W2 = incidence(G);
disp('关联矩阵 (W2) 的稀疏表示:');
disp(full(W2)); % 使用 full() 查看完整的稠密矩阵
