| **命令名称**               | **功 能**                     | **基本用途和语法**                                                         |
| ---------------------- | --------------------------- | ------------------------------------------------------------------- |
| **`distances`**        | 求图中所有顶点对之间的最短距离。            | **用途：** 用于计算图的**所有对最短路径 (All-Pairs Shortest Path, APSP)** 矩阵。       |
|                        |                             | **语法：** `D = distances(G)`                                          |
|                        |                             | _D_ 是一个矩阵，$D(i, j)$ 表示从节点 $i$ 到节点 $j$ 的最短路径距离。                      |
| **`conncomp`**         | 找无向图的连通分支，或有向图的强(弱)连通分支。    | **用途：** 分析图的连通性，将图划分成独立的、不可达的子图。                                    |
|                        |                             | **语法：** `[bin, binsize] = conncomp(G)`                              |
|                        |                             | _bin_ 是一个向量，指示每个节点属于哪个连通分支的 ID。                                     |
| **`isdag`**            | 测试有向图是否含有圈，不含圈则返回 1，否则返回 0。 | **用途：** 判断有向图是否是**有向无环图 (DAG)**。DAG 在调度、拓扑排序中非常重要。                  |
|                        |                             | **语法：** `tf = isdag(G)`                                             |
|                        |                             | _tf_ 为逻辑值 (`true` 或 `false` / 1 或 0)。                               |
| **`isomorphism`**      | 确定两个图是否同构。                  | **用途：** 判断两个图在拓扑结构上是否完全相同（忽略节点和边的标签差异）。                             |
|                        |                             | **语法：** `tf = isomorphism(G1, G2)`                                  |
|                        |                             | _tf_ 为逻辑值 (`true` 或 `false` / 1 或 0)。                               |
| **`maxflow`**          | 计算有向图的最大流。                  | **用途：** 求解网络流问题，计算从源点到汇点的最大传输能力。                                    |
|                        |                             | **语法：** `[M, F] = maxflow(G, s, t)`                                 |
|                        |                             | _M_ 是最大流量的值，_F_ 是一个图对象，其边权重为计算出的流值。                                 |
| **`minspantree`**      | 在图中求最小生成树。                  | **用途：** 求解连通无向图的最小生成树 (MST) 问题。                                     |
|                        |                             | **语法：** `T = minspantree(G)`                                        |
|                        |                             | _T_ 是一个图对象，包含原图 _G_ 的最小生成树的边。                                       |
| **`reordernodes`**     | 对图顶点重新排序。                   | **用途：** 优化图的内部存储顺序，有时用于提高某些算法的效率，或按特定标准组织节点。                        |
|                        |                             | **语法：** `Gnew = reordernodes(G, order)`                             |
|                        |                             | _order_ 是一个包含新的节点顺序的向量或元胞数组。                                        |
| **`shortestpath`**     | 求图中指定的**一对顶点**间的最短距离和最短路径。  | **用途：** 求解**单源单汇最短路径**问题，最常用。                                       |
|                        |                             | **语法：** `[path, d] = shortestpath(G, s, t)`                         |
|                        |                             | _path_ 是最短路径上的节点序列，_d_ 是最短距离。                                       |
| **`shortestpathtree`** | 求顶点的最短路径树。                  | **用途：** 求解**单源最短路径 (Single-Source Shortest Path, SSSP)** 问题，结果是一棵树。 |
|                        |                             | **语法：** `T = shortestpathtree(G, s)`                                |
|                        |                             | _T_ 是一个图对象，包含从源点 _s_ 到所有可达节点的最短路径构成的树。                              |
| **`subgraph`**         | 提出子图。                       | **用途：** 根据给定的节点或边集合，从原图 _G_ 中提取出一个子图。                               |
|                        |                             | **语法：** `H = subgraph(G, N)`                                        |
|                        |                             | _H_ 是由 _G_ 中节点集合 _N_ 及其所有关联边构成的子图。                                  |
